# Code of Conduct

## Overview

This is a personal portfolio repository maintained by Kayan Shah. Contributions, issues, and interactions are welcome, provided they remain respectful and constructive.

## Attribution Requirement

Any use, modification, distribution, or derivative work based on any part of this codebase — in whole or in part — must clearly credit the original author:

**Kayan Shah** — [github.com/KayanShah](https://github.com/KayanShah)

This credit must be:
- Visible and prominent (not hidden in comments or footnotes)
- Present in any public-facing project, README, or documentation that uses or builds upon this code
- Retained in all copies or substantial portions of the code

Failure to provide attribution is a violation of this Code of Conduct and the terms under which this code is made available.

## Expected Behaviour

- Be respectful and considerate in all interactions
- Provide constructive feedback on issues or pull requests
- Always credit Kayan Shah when using or referencing this code

## Unacceptable Behaviour

- Harassment, abuse, or discriminatory language
- Spam or unsolicited self-promotion
- Misrepresenting or impersonating the repository owner
- Using, copying, or modifying this code without proper attribution to Kayan Shah

## Reporting

If you experience or witness unacceptable behaviour, please contact [kayan.shah@icloud.com](mailto:kayan.shah@icloud.com).

## Scope

This Code of Conduct applies to all interactions within this repository and to any use of this code outside of it.
